# Authentication System

A full-stack authentication system built using React, Node.js, Express.js, MongoDB Atlas, JWT, Vercel and Render.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Dashboard
- Profile Fetch
- Logout Functionality
- Password Visibility Toggle
- Loading State Handling
- Forgot Password
- MongoDB Atlas Integration

## Tech Stack

Frontend:
- React
- React Router

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Deployment:
- Vercel (Frontend)
- Render (Backend)
